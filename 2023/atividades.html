MC01: Git e GitHub: Do Básico ao Avançado

Caio Henrique Lopes Sousa, Eduardo Paulino Dantas e Halan Caio Pereira do Nascimento.

Objetivo: Introduzir os participantes às ferramentas Git e GitHub.


Justificativa: Este minicurso é essencial porque aborda ferramentas fundamentais para qualquer desenvolvedor. Ao longo do programa, os participantes terão a oportunidade de familiarizar-se com conceitos básicos e avançados de Git e GitHub, ambos amplamente utilizados na prática diária. Assim, visa-se proporcionar uma introdução abrangente a essas ferramentas cruciais para o desenvolvimento de software.


Pré-requisitos: Não são necessários conhecimentos prévios.

MC02: Programação C# para iniciantes

Rebeca Barbosa Lourenço e Gabriel Lira da Nóbrega.

Objetivo: Introduzir a linguagem C# na Universidade.


Justificativa: Introduzir a linguagem C# na faculdade é uma iniciativa importante, pois amplia as opções de carreira para os alunos e preenche uma lacuna muitas vezes negligenciada. Como uma linguagem robusta e amplamente utilizada, o conhecimento em C# pode proporcionar aos estudantes oportunidades em uma variedade de setores, incluindo desenvolvimento de software, jogos, aplicativos web e móveis.


Pré-requisitos: Lógica de programação.

MC07: Introdução à Programação Web: Guia Básico em HTML, CSS e JavaScript

Yuri dos Santos Monteiro

Objetivo: Proporcionar aos alunos uma compreensão fundamental e prática dos conceitos básicos de desenvolvimento web, abrangendo HTML, CSS e JavaScript, preparando-os para construir suas próprias aplicações web simples.


Justificativa: Devido à alta demanda por programadores web no mercado atual, é crucial adquirir habilidades nessa área. No entanto, algumas universidades oferecem aulas de programação web apenas nos períodos finais do curso. Nesse contexto, cursos complementares, se torna uma estratégia valiosa para antecipar e fortalecer as habilidades necessárias para atender às demandas do mercado de trabalho.


Pré-requisitos: Sem pré-requisitos.

OF01: LinkedIn na Prática: Estratégias e Oportunidades para Impulsionar sua Carreira

Ellen Santos Ramalho, Emilly Santos Ramalho, Rebeca Barbosa Lourenço e Yohanna Barbara Pereira Costa

Objetivo: Mostrar aos participantes a oportunidade de alavancar suas carreiras por meio do LinkedIn, fornecendo estratégias eficazes para aumentar sua visibilidade, expandir sua rede e explorar oportunidades profissionais na plataforma.


Justificativa: o LinkedIn se tornou um ponto central no mundo profissional, funcionando como um verdadeiro cartão de visitas virtual. Pré-requisitos: Não tem nenhum pré-requisito.
